Negrito / Destaque

Nesta frase, temos um trecho em destaque usando a tag de semantica 'strong'

Itálico / Ênfase

Nesta frase, temos um trecho em itálico usando a tag de forma 'i'

Texto marcado

Podemos também criar um trecho marcado usando a tag MARK

Texto grande e pequeno

Estamos criando um trecho grande e um trecho pequeno nesse paragrafo

Texto deletado

Podemos marcar um texto como excluido para indicar que ele pode ser lido, mas nao considerado

Texto inserido

Podemos marcar um texto inserido para dar uma enfase e indicar que ele foi adicionado depois.

Existe tambem o sublinhado, com a tag de forma 'U'

Texto sobrescrito

Para inserir coisas do tipo x^20 + 3 usando a tag 'SUP' -> x20 + 3

Texto subscrito

Para inserir coisas do tipo H2O usando a tag 'SUB' -> H2O
