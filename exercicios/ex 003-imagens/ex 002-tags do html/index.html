Parágrafos e quebra de linha

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags p e \p.

Para fazer a quebra de linha em algum lugar específico,
basta escrever a tag br nesse local.

Já para escrever tags como maior e menor no seu site,
basta usar <p> </p>

Para fazer um comentário, se utiliza
<!-- + Enter
